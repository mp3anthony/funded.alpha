## Agent skills

### Issue tracker

Issues and PRDs are tracked as GitHub issues on `mp3anthony/funded` via the `gh` CLI. See `docs/agents/issue-tracker.md`.

### Triage labels

The canonical triage roles, each label string equal to its name (`needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `ready-for-testing`, `wontfix`). See `docs/agents/triage-labels.md`.

### Domain docs

Single-context: glossary at `docs/GLOSSARY.md`, decision records under `docs/decisions/`. See `docs/agents/domain.md`.
